Function Documentation

float acos ( float  num)

The acos function uses a lookup table for corresponding output. Output data are stored as byte values (0 - 255), they are scaled down to float number (0.0 - 1.0) for output.

float sin ( int  deg)

The functions for sin and cos use lookup table to determine the sin or cos value of the input angle. Input for these functions are scaled up 10 times. e.g. -450 = -45.0 deg

  • Both functions return a value between -1 and 1. (e.g. input: -450, output -> -0.7071)

  • 0 to 0.9 is in steps of 0.0079 rads. (1/127)
  • 0.9 to 0.99 is in steps of 0.0008 rads. (0.01/127)
  • 0.99 to 1 is in steps of 0.0002 rads. (0.01/64)
